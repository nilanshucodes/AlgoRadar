Privacy Policy

Last updated: December 16, 2024

Short version: We don't collect your data. That's it. This privacy policy is provided to comply with Chrome Web Store requirements.

What This Covers

This privacy policy applies to both:

  • AlgoRadar website (what you're on right now)
  • AlgoRadar Chrome extension (if you have it installed)

Data Collection

Website: We don't use cookies, trackers, or analytics.Our hosting provider (Vercel) may collect basic server logs such as page requests. This does not apply to the Chrome extension.

Chrome Extension: Stores contest data locally on your computer so it loads faster. This data never leaves your device. We don't see it, track it, or want it.

Contact Form: If you fill out our contact form, we obviously need your email to reply to you. We store your message in our database and that's it. Contact form data is used only to respond to user inquiries and is never used for marketing.

What the Extension Does

The Chrome extension needs a few permissions to work:

  • Storage: Saves contest info on your computer so it doesn't have to reload everything each time
  • Alarms: Auto-refreshes contest data every 20 minutes so you see new contests
  • Network Access: Connects to our server to get contest data from CLIST API

That's literally all it does. No shady stuff.

Third-Party Services

CLIST: We get all contest data from clist.by. Their API is public and we don't send them any info about you.

Vercel: Hosts our website and backend. They handle the infrastructure but don't track users beyond basic server logs.

Your Data Rights

Since we barely collect anything, there's not much to do here. But:

  • Contact form messages? Email us and we'll delete them
  • Extension data? Just uninstall it and everything's gone
  • Want proof? Check our code on GitHub

Updates to This Policy

If we ever change how we handle data (unlikely), we'll update this page and let you know. We're not planning to suddenly start tracking users or anything weird like that.

Questions?

Hit us up:

  • Contact form: Right here
  • GitHub issues: Report bugs or ask questions

TL;DR: We're developers who hate data collection as much as you do. AlgoRadar exists to help you not miss contests. That's it.

This policy is designed to comply with GDPR, CCPA, and Chrome Web Store policies. because we literally don't collect anything to worry about.
